VS Code 必备扩展推荐#

简介#

VS Code 是目前最受欢迎的代码编辑器之一，其强大的扩展生态系统让开发者能够根据需求定制开发环境。本文推荐一些实用的扩展插件。

代码高亮和语法支持#

1. Material Icon Theme#

  • 功能: 为不同类型的文件提供美观的图标
  • 安装: 在扩展市场搜索 “Material Icon Theme”
  • 推荐指数: ⭐⭐⭐⭐⭐

2. Bracket Pair Colorizer#

  • 功能: 用不同颜色显示匹配的括号对
  • 安装: 在扩展市场搜索 “Bracket Pair Colorizer”
  • 推荐指数: ⭐⭐⭐⭐

3. Auto Rename Tag#

  • 功能: 自动重命名配对的 HTML/XML 标签
  • 安装: 在扩展市场搜索 “Auto Rename Tag”
  • 推荐指数: ⭐⭐⭐⭐⭐

代码格式化#

4. Prettier - Code formatter#

  • 功能: 自动格式化代码，支持多种语言
  • 安装: 在扩展市场搜索 “Prettier”
  • 推荐指数: ⭐⭐⭐⭐⭐

5. Beautify#

  • 功能: 格式化 HTML, CSS, JavaScript, JSON
  • 安装: 在扩展市场搜索 “Beautify”
  • 推荐指数: ⭐⭐⭐⭐

智能提示和补全#

6. IntelliSense for CSS class names in HTML#

  • 功能: 在 HTML 中提供 CSS 类名的智能提示
  • 安装: 在扩展市场搜索 “IntelliSense for CSS”
  • 推荐指数: ⭐⭐⭐⭐

7. Path Intellisense#

  • 功能: 自动补全文件路径
  • 安装: 在扩展市场搜索 “Path Intellisense”
  • 推荐指数: ⭐⭐⭐⭐⭐

Git 集成#

8. GitLens#

  • 功能: 增强 Git 功能，显示代码行的作者信息
  • 安装: 在扩展市场搜索 “GitLens”
  • 推荐指数: ⭐⭐⭐⭐⭐

9. Git History#

  • 功能: 查看文件的历史记录和比较
  • 安装: 在扩展市场搜索 “Git History”
  • 推荐指数: ⭐⭐⭐⭐

主题和外观#

10. One Dark Pro#

  • 功能: 基于 Atom 的 One Dark 主题
  • 安装: 在扩展市场搜索 “One Dark Pro”
  • 推荐指数: ⭐⭐⭐⭐⭐

11. Dracula Official#

  • 功能: 经典的 Dracula 暗色主题
  • 安装: 在扩展市场搜索 “Dracula Official”
  • 推荐指数: ⭐⭐⭐⭐

实用工具#

12. Live Server#

  • 功能: 启动本地开发服务器，支持实时重载
  • 安装: 在扩展市场搜索 “Live Server”
  • 推荐指数: ⭐⭐⭐⭐⭐

13. REST Client#

  • 功能: 在 VS Code 中测试 REST API
  • 安装: 在扩展市场搜索 “REST Client”
  • 推荐指数: ⭐⭐⭐⭐

14. Thunder Client#

  • 功能: 轻量级的 API 测试工具
  • 安装: 在扩展市场搜索 “Thunder Client”
  • 推荐指数: ⭐⭐⭐⭐⭐

配置建议#

设置文件#

在 VS Code 的 settings.json 中添加以下配置：

{
  "editor.formatOnSave": true,
  "editor.formatOnPaste": true,
  "editor.tabSize": 2,
  "editor.insertSpaces": true,
  "files.autoSave": "afterDelay",
  "files.autoSaveDelay": 1000
}

快捷键#

  • Ctrl + Shift + P: 打开命令面板
  • Ctrl + ,: 打开设置
  • Ctrl + Shift + X: 打开扩展面板
  • Ctrl + K Ctrl + S: 打开键盘快捷键设置

总结#

这些扩展能够显著提升你的开发效率。建议根据你的具体需求选择合适的扩展，避免安装过多不必要的插件影响性能。
